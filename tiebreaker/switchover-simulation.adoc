= MetroCluster Tiebreaker를 사용하여 전환을 시뮬레이션합니다.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Tiebreaker 1.7 버전부터는 MetroCluster Tiebreaker의 전환 기능을 테스트하기 위해 전환 시뮬레이션을 수행할 수 있습니다.

.이 작업에 대해
* 그만큼 `monitor switchover-simulate` 이 명령어는 Tiebreaker 1.7 이상 버전에서만 지원됩니다.


.단계
. 타이브레이커 모니터링 상태를 확인하세요:
`monitor show -status`

. 전환 시뮬레이션을 수행하십시오.
+
`monitor switchover-simulate -monitor-name <monitor_name> -cluster <cluster_name>`

+
명령이 성공적으로 실행되면 다음과 같은 출력이 표시됩니다.

+
[listing]
----
Successfully triggered Switchover Simulation. Please check the status of the Switchover Simulation on the ONTAP cluster using command "metrocluster operation history show"
----
