[NOTE]
====
* 값이 누락되면 플랫폼을 MetroCluster 호환 스위치와 함께 사용할 수 없음을 나타냅니다.
* AFF A30, AFF C30, AFF C60 및 FAS50 시스템은 25Gbps 네트워크 속도를 지원하기 위해 컨트롤러 카드에 QSFP-to-SFP 어댑터가 필요합니다.
